# Knowledge Graph

## Objectif

Le Knowledge Graph est le cœur de l'intelligence de Rossiyani.

Il centralise toutes les connaissances linguistiques de la plateforme afin de proposer une expérience rapide, cohérente et évolutive.

Son rôle est de transformer chaque analyse en connaissance durable réutilisable dans tout le produit.

---

# Philosophie

Rossiyani ne repose pas sur des appels permanents à l'intelligence artificielle.

Les analyses sont produites une fois, enrichies progressivement, puis réutilisées autant de fois que nécessaire.

Cette approche garantit :

* une expérience instantanée ;
* des coûts maîtrisés ;
* une qualité homogène ;
* une amélioration continue de la plateforme.

---

# Les connaissances stockées

Le Knowledge Graph regroupe notamment :

* les mots ;
* les lemmes ;
* les flexions ;
* les cas grammaticaux ;
* les conjugaisons ;
* les constructions ;
* les expressions ;
* les collocations ;
* les phrases ;
* les textes ;
* les concepts grammaticaux.

Toutes ces informations sont reliées entre elles.

---

# Une source unique de vérité

Le Knowledge Graph constitue la référence linguistique de Rossiyani.

Toutes les fonctionnalités utilisent les mêmes données afin de garantir des explications cohérentes dans l'ensemble de l'application.

Reader, Vocabulary, Review et Compose ne doivent jamais maintenir leurs propres connaissances séparément.

---

# Réutilisation

Une analyse produite pour un texte peut être réutilisée dans :

* Vocabulary ;
* Review ;
* Compose ;
* les recommandations ;
* les statistiques ;
* les futures fonctionnalités.

Chaque nouvelle donnée enrichit progressivement l'ensemble de l'écosystème.

---

# Performance

Le Knowledge Graph privilégie le pré-calcul.

Les appels à l'IA sont réservés :

* aux nouveaux contenus ;
* aux enrichissements ;
* aux cas exceptionnels.

Toutes les données déjà connues doivent être réutilisées.

---

# Évolutivité

Le Knowledge Graph doit pouvoir évoluer sans remettre en cause l'architecture du produit.

De nouveaux types de connaissances pourront être ajoutés progressivement sans modifier les fonctionnalités existantes.

---

# Objectif long terme

À long terme, le Knowledge Graph doit devenir une base de connaissances linguistiques suffisamment riche pour alimenter l'ensemble des fonctionnalités de Rossiyani.

Chaque texte analysé, chaque mot étudié et chaque correction réalisée contribuent à enrichir cette base.

Le Knowledge Graph représente l'actif principal du produit et constitue son principal avantage concurrentiel.
