# Vocabulary pattern integration

Links saved words to Learning Patterns via:

- **Pattern instances** in sentences containing the lemma
- **Knowledge concept keys** matched against pattern `knowledgeConceptKeys`

Used by `buildVocabularyWordFiche()` → `patternSlice` on the word fiche.

Depth on pattern cards is decided client-side through `explore_vocabulary` on the Learning Orchestrator (same encounter store as the Reader).
