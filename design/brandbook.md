# ROSSIYANI — BRANDBOOK

Version 2.0

This document is the canonical source of truth for Rossiyani's identity.

If any existing interface, component or implementation conflicts with this document, this document always wins.

---

# 1. VISION

Rossiyani exists to make Russian understandable.

Not easier.

Not gamified.

Understandable.

Rossiyani teaches users to see the internal logic of the language instead of memorizing isolated rules.

Every page should answer one question:

> "Why do Russians express this idea this way?"

Rossiyani is an exploration environment built around relationships between words, concepts, grammar and authentic texts.

---

# 2. MISSION

Create the best environment in the world for deeply understanding Russian.

Every word is connected.

Every form belongs to a lemma.

Every lemma belongs to concepts.

Every concept links to explanations.

Every explanation links back to authentic texts.

The product is a living linguistic graph.

---

# 3. POSITIONING

Rossiyani is NOT:

* Duolingo
* LingQ
* Anki
* Memrise
* a dictionary
* an AI chatbot
* a dashboard
* a SaaS product

Rossiyani IS:

* an editorial product
* a linguistic explorer
* a research notebook
* an encyclopedia
* a reading companion
* a knowledge graph

---

# 4. CORE IDEA

Rossiyani should feel like discovering the private notebook of an exceptional Russian professor.

Inside are:

* notes
* diagrams
* annotations
* references
* cross-links
* etymology
* grammar observations
* examples
* exceptions

Nothing feels generated.

Everything feels carefully written.

---

# 5. DESIGN PHILOSOPHY

The interface should disappear.

Reading should dominate.

Understanding should dominate.

Navigation should feel effortless.

Every visual decision must improve comprehension.

Never decorate information.

Reveal information.

# 13. TYPOGRAPHY PHILOSOPHY

Typography is the identity.

Large editorial serif titles create emotion.

Neutral sans-serif creates clarity.

Monospace creates linguistic precision.

Hierarchy comes from typography first.

Not containers.

---

thin

minimal

functional

quiet

editorial

Never decorative.

Never oversized.

Never colorful.

---

# 15. ANIMATIONS

Animation should never attract attention.

No bounce.

No glow.

No spring effects.

No exaggerated easing.

Movement simply accompanies reading.

Fade.

Slide.

Expand.

Nothing else.

---

# 20. FINAL TEST

Before shipping any interface, ask:

Does this look like software?

If yes,

simplify.

Does this look like an editorial linguistic explorer?

If yes,

Rossiyani is moving in the right direction.
