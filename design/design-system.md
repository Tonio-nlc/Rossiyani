# Rossiyani Design System v2

## Product Identity

Rossiyani is a premium language learning workspace built around reading.

The product is not a digital library.

The product is not an academic portal.

The product is a personal language-learning operating system.

Every screen must help the learner decide:

"What should I do next?"

---

# Interface Architecture

Rossiyani uses two visual systems.

## Workspace System

Used for:

* Home
* Explorer
* Practice
* Vocabulary
* Profile
* Future analytics

Characteristics:

* Large cards
* Clear hierarchy
* Visible progress
* Strong actions
* Comfortable spacing
* Dashboard feeling

Inspiration:

* Readwise
* Linear
* Duolingo Super
* Stitch dashboard concepts

---

# Success Test

If a screen feels like:

* an academic portal
* a university website
* a document archive

The design failed.

If a screen feels like:

* a premium learning workspace
* a modern SaaS product
* a focused learning dashboard

The design succeeded.
