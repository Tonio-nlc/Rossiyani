# README_PROJECT.md

# Project Overview

Russian Deep Reading is an educational web application designed for French-speaking learners of Russian.

The goal is not translation.

The goal is deep understanding of authentic Russian.

The application must explain:

* grammar
* declensions
* conjugations
* stress patterns
* word order
* native expressions
* Russian logic
* differences between Russian and French thinking patterns

The application should help users understand why every element exists inside a sentence.

# Development Philosophy

The project is expected to evolve for years.

Maintainability is more important than development speed.

Readability is more important than cleverness.

Predictability is more important than optimization.

---

# Primary Product

The primary product is NOT the reader.

The primary product is the linguistic analysis engine.

The reader is simply a way to display the analysis.

All architectural decisions must prioritize analysis quality.

---

# Educational Philosophy

The application teaches through:

* authentic content
* sentence analysis
* word analysis
* pattern recognition

The application does not teach through:

* gamification
* streaks
* points
* badges

Learning quality has priority over engagement tricks.

---

# Target User

Native French speaker.

Learning Russian from beginner to advanced level.

Interested in understanding how Russian works.

Not interested in memorizing rules without context.
