# Writing Guidelines

## Objectif

Ce document définit les règles éditoriales de Rossiyani.

Tous les textes de la bibliothèque doivent respecter ces principes afin de garantir une expérience cohérente, progressive et adaptée aux apprenants.

L'objectif est de créer une bibliothèque originale, pédagogique et agréable à lire.

---

# Philosophie

Chaque texte doit raconter une histoire ou présenter une situation crédible.

L'utilisateur ne lit pas pour faire un exercice.

Il lit parce que le contenu est intéressant.

La pédagogie est intégrée naturellement dans le récit.

---

# Authenticité

Les textes doivent ressembler à du russe réellement utilisé.

Les dialogues doivent paraître naturels.

Les formulations artificielles ou traduites littéralement depuis le français sont interdites.

---

# Progression

Chaque texte doit introduire un nombre limité de nouveautés.

La majorité du vocabulaire doit rester connue de l'apprenant.

Les nouvelles notions sont toujours réutilisées dans les textes suivants.

La difficulté augmente progressivement au sein de chaque collection.

---

# Longueur

Les textes restent volontairement courts.

Ils doivent pouvoir être lus en quelques minutes.

Le temps de lecture augmente progressivement avec le niveau.

---

# Collections

Chaque collection poursuit un objectif pédagogique clair.

Exemples :

* Vie quotidienne
* Premières conversations
* Voyages
* Travail
* Université
* Culture russe
* Histoires courtes

Les textes d'une même collection doivent former un parcours logique.

---

# Niveaux

Chaque texte est associé à un niveau CECRL.

Les critères sont définis principalement par :

* le vocabulaire ;
* les structures grammaticales ;
* la longueur ;
* la complexité syntaxique.

---

# Vocabulaire

Le vocabulaire doit être fréquent et utile.

Les mots rares sont introduits uniquement lorsqu'ils apportent une véritable valeur pédagogique.

Les répétitions sont encouragées lorsqu'elles facilitent la mémorisation.

---

# Grammaire

Chaque texte met naturellement en avant certaines notions.

La grammaire n'est jamais présentée sous forme de leçon.

Elle apparaît dans un contexte réel.

---

# Culture

Lorsque cela est pertinent, les textes introduisent des éléments de culture russe.

Ces informations restent courtes et directement liées au récit.

---

# Style

Le style est simple.

Les phrases sont naturelles.

Les personnages sont crédibles.

Les dialogues doivent pouvoir être entendus dans la vie quotidienne.

---

# Interdictions

Les textes ne doivent jamais :

* être générés sans cohérence pédagogique ;
* accumuler inutilement les difficultés ;
* traduire mot à mot le français ;
* ressembler à un manuel scolaire ;
* être créés uniquement pour illustrer une règle de grammaire.

---

# Objectif final

La bibliothèque Rossiyani doit permettre à un apprenant de progresser naturellement en lisant des textes de qualité.

Chaque texte doit donner envie de lire le suivant.

L'expérience globale doit se rapprocher davantage d'une collection de nouvelles que d'une succession d'exercices.
