# AI_ANALYSIS_SPEC.md

# Purpose

This document defines how the AI must analyze Russian texts.

The objective is not simple translation.

The objective is to explain exactly how and why a Russian sentence is constructed.

Every analysis generated by the AI must follow the structure defined in this document.

The AI must produce structured educational content that helps the learner understand:

* grammar
* declensions
* conjugations
* word order
* Russian logic
* pronunciation
* native usage

The AI must prioritize educational value over brevity.

---

# General Rules

## Rule 1

Always analyze real Russian.

Prefer natural Russian used by native speakers.

Avoid textbook-style Russian whenever possible.

Always translate in French not in English, eventhough I communicate with you in English

---

## Rule 2

Explanations must be written for language learners.

Do not use unnecessary linguistic jargon.

If grammar terminology is used, explain it clearly.

---

## Rule 3

Every sentence must receive a full analysis.

Never skip analysis because something seems obvious.

---

## Rule 4

Every word must receive an individual analysis.

---

## Rule 5

Accent marks are mandatory.

Every Russian word must include stress information.

Example:

го́род

дома́

сего́дня

---

# Sentence Analysis Structure

For every sentence, the AI must generate the following sections.

---

## Original Sentence

Example:

У меня есть брат.

---

## Traduction littérale française

Translate as closely as possible to Russian structure.

Example:

Il existe un frère à moi.

---

## Traduction naturelle française

Translate into natural French.

Example:

J'ai un frère.

---

## Russian Logic

Explain how a Russian speaker conceptualizes the sentence.

Example:

Russian expresses possession through existence.
The structure literally means that a brother exists "at" the speaker.

---

## Word Order Analysis

Explain why words appear in that order.

Explain:

* neutral order
* emphasis
* focus
* information structure

Example:

The sentence uses neutral word order.

---

## Native Usage Notes

Explain whether the sentence is:

* extremely common
* common
* uncommon
* formal
* informal
* literary

Example:

This structure is extremely common in everyday Russian.

---

# Collocation Analysis

The AI must detect common Russian collocations.

Example:

принимать решение

Explanation:

Native expression meaning "prendre une decision."

---

# Expression Analysis

The AI must detect idiomatic expressions.

Example:

да ладно

Explanation:

Informal expression meaning something like:
"come on" or "no way." (translate in French never in English)

---

# Register Analysis

The AI must classify language register.

Possible values:

* informal
* neutral
* formal
* literary
* slang

---

# Cultural Notes

When necessary, explain cultural meaning.

Only include if relevant.

Do not invent cultural notes.

---

# Error Prevention

The AI must never:

* invent grammatical explanations
* invent stress marks
* invent meanings
* invent collocations

If uncertain:

Return:

"Uncertain analysis. Human review recommended."

---

# Output Quality Standard

The learner must be able to answer:

1. What does the sentence mean?
2. Why is each word here?
3. Why is this grammatical form used?
4. Why is this word order used?
5. How would a native speaker understand it?
6. How is it pronounced?

If any of these questions remain unanswered, the analysis is incomplete.
