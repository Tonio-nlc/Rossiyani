# Vocabulary

## Objectif

Vocabulary est la base de connaissances linguistique de Rossiyani.

Chaque mot rencontré pendant la lecture possède une fiche dédiée qui centralise toutes les informations nécessaires pour le comprendre, le mémoriser et le réutiliser.

Vocabulary ne doit jamais être un simple dictionnaire. Son objectif est d'expliquer comment un mot fonctionne dans la langue russe, toujours en lien avec les textes lus.

---

# Philosophie

Vocabulary répond à une question simple :

> "Je viens de rencontrer ce mot. Que dois-je savoir pour vraiment le comprendre ?"

Toutes les informations doivent être utiles à l'apprentissage.

L'utilisateur ne doit jamais être noyé sous des détails linguistiques inutiles.

---

# Structure de la page

## 1. En-tête

Affiche les informations essentielles :

* Mot
* Prononciation
* Audio
* Traduction principale
* Niveau estimé (A1 à C2)
* Fréquence d'utilisation
* Catégorie grammaticale

---

## 2. Comprendre

Cette section explique immédiatement le mot.

Elle contient :

* définition principale
* traductions possibles
* nuances de sens
* remarques importantes
* faux amis éventuels

---

## 3. Fonctionnement grammatical

Selon la nature du mot, cette section affiche :

Pour un verbe :

* aspect
* infinitif
* conjugaison
* verbes associés

Pour un nom :

* genre
* animé/inanimé
* déclinaisons
* pluriel

Pour un adjectif :

* formes
* déclinaisons
* comparatif
* superlatif

Pour les autres catégories :

* informations grammaticales pertinentes

---

## 4. Exemples

Exemples authentiques provenant de la bibliothèque Rossiyani.

Chaque exemple est accompagné :

* d'une traduction ;
* d'un audio ;
* d'un lien vers le texte d'origine.

---

## 5. Expressions

Liste des expressions courantes contenant ce mot.

Chaque expression est expliquée avec son sens réel.

---

## 6. Famille de mots

Vocabulary présente les mots liés :

* dérivés
* préfixes
* suffixes
* mots de la même famille

L'objectif est de montrer les liens entre les mots plutôt que de les apprendre isolément.

---

## 7. Liens linguistiques

Cette section connecte le mot au Knowledge Graph.

Elle peut afficher :

* constructions grammaticales
* collocations
* synonymes
* antonymes
* verbes liés
* concepts associés

---

## 8. Révision

Depuis cette page, l'utilisateur peut :

* ajouter le mot aux révisions ;
* retirer le mot des révisions ;
* le marquer comme difficile ;
* consulter son historique d'apprentissage.

---

# Navigation

Depuis n'importe quelle fiche, l'utilisateur peut accéder facilement aux mots liés afin d'explorer naturellement la langue.

Vocabulary doit encourager la curiosité.

---

# Intégration avec le Reader

La majorité des visites sur Vocabulary proviennent du Reader.

Le passage entre les deux pages doit être instantané.

L'utilisateur ne doit jamais avoir l'impression de quitter sa lecture.

---

# Version mobile

La structure reste identique mais les sections deviennent repliables afin de faciliter la navigation.

Les informations essentielles restent toujours visibles en premier.

---

# Évolutions futures

Vocabulary pourra progressivement intégrer :

* cartes de fréquence ;
* évolution personnelle de la maîtrise du mot ;
* mots souvent confondus ;
* recommandations de vocabulaire ;
* visualisation des familles de mots ;
* liens vers les règles grammaticales concernées.

Chaque évolution devra conserver un principe fondamental : faire de Vocabulary un compagnon de lecture, jamais un dictionnaire encyclopédique intimidant.
