# Review

## Objectif

Review est le système de mémorisation de Rossiyani.

Son rôle est d'aider l'utilisateur à retenir durablement le vocabulaire, les expressions et les constructions rencontrés pendant ses lectures.

Review ne remplace pas la lecture. Il la complète.

---

# Philosophie

Chaque révision provient d'un contexte réel.

Rossiyani ne crée pas des listes de mots déconnectées.

L'utilisateur révise uniquement ce qu'il a réellement rencontré dans ses textes.

L'objectif est de favoriser une mémorisation naturelle grâce à la répétition espacée.

---

# Fonctionnement

Review utilise un système de répétition espacée (SRS).

Chaque mot, expression ou construction suit un calendrier de révision qui évolue selon les réponses de l'utilisateur.

Les éléments bien maîtrisés apparaissent moins souvent.

Les éléments difficiles reviennent plus rapidement.

---

# Types de cartes

## Vocabulaire

Reconnaître un mot.

Retrouver son sens.

Écouter sa prononciation.

---

## Expressions

Comprendre une expression complète.

Reconnaître son sens réel.

La replacer dans son contexte.

---

## Constructions grammaticales

Identifier une structure.

Comprendre son fonctionnement.

Reconnaître quand l'utiliser.

---

# Déroulement d'une session

Une session de révision doit être rapide.

L'utilisateur lance Review.

Rossiyani présente les cartes du jour.

À la fin de la session, les prochaines révisions sont automatiquement planifiées.

L'utilisateur peut ensuite retourner naturellement à la lecture.

---

# Création des cartes

Les cartes sont générées automatiquement à partir de :

* la bibliothèque de textes ;
* l'activité de lecture ;
* les mots enregistrés ;
* les éléments marqués comme difficiles.

L'utilisateur peut également ajouter manuellement un mot ou une expression depuis Vocabulary ou le Reader.

---

# Progression

Review affiche uniquement les informations utiles :

* cartes à réviser aujourd'hui ;
* cartes maîtrisées ;
* nouvelles cartes disponibles ;
* progression globale.

Les statistiques restent volontairement simples.

---

# Expérience utilisateur

Une session de révision doit durer quelques minutes.

L'interface doit être rapide, claire et sans distraction.

Le but est d'encourager une pratique quotidienne sans transformer Rossiyani en application de flashcards.

---

# Version mobile

Toutes les interactions doivent pouvoir être réalisées d'une seule main.

Les réponses sont rapides et adaptées aux petits écrans.

---

# Données utilisées

Review s'appuie sur :

* la progression utilisateur ;
* les mots rencontrés ;
* les expressions apprises ;
* les constructions étudiées ;
* l'historique des révisions.

---

# Évolutions futures

Review pourra progressivement intégrer :

* révisions audio ;
* dictées ;
* production écrite ;
* révisions personnalisées selon les difficultés ;
* cartes générées automatiquement à partir des dernières lectures.

Quelle que soit son évolution, Review devra toujours rester un complément à la lecture, jamais une activité dominante.
