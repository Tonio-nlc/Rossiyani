## Comment dire « poser », « mettre » (un objet) en russe ?

**Класть** (poser — processus) et **положить** (poser — une fois, résultat) servent pour ranger, déposer, mettre sur une surface.

### Le mécanisme en une phrase

**Класть / положить + accusatif** = mettre quelque part (horizontal).

**Exemple :** **Кладу книгу на стол.** — Je pose le livre sur la table.

**Exemple :** **Положи ключи сюда.** — Mets les clés ici.

**Exemple :** **Он положил документы в папку.** — Il a mis les documents dans le dossier.

**Contre-exemple :** **Каждый раз положил на стол** — répétition : **кладу**.

────────────────────────

Situation :

« On te demande de déposer tes affaires à un endroit précis avant d'entrer. »

↓

**Пожалуйста, положите сумку на полку.** — S'il vous plaît, posez votre sac sur l'étagère.

────────────────────────

Pourquoi ?

**Класть / положить** vs **ставить / поставить** (vertical) — distinction spatiale importante.

```
  horizontal (класть)          vertical (ставить)
  ───────────────────          ─────────────────
  положить на стол             поставить на стол
  класть в сумку               поставить вазу
  положить документы           поставить чашку
```

────────────────────────

💡 Astuce

**Положить** = action terminée · **Класть** = en train de poser / habitude.

**Ставить / поставить** = mettre debout.

────────────────────────

⭐ Très fréquent

**Положи сюда!** · **Куда положить?** · **Я положил в сумку.**

────────────────────────

⚠ Attention

**Класть** ≠ **ложить** (vieilli/fautif) — toujours **класть**.

────────────────────────

### Conjugaison (класть)

| Personne | Forme |
|----------|-------|
| я | кладу |
| ты | кладёшь |
| он/она | кладёт |
| мы | кладём |
| вы | кладёте |
| они | кладут |

**Impératif perf. :** **положи** · **положите**

### Phrases utiles

| Russe | Français |
|-------|----------|
| Куда мне положить? | Où est-ce que je pose ? |
| Положи это на стол. | Pose ça sur la table. |
| Я кладу деньги в карман. | Je mets l'argent dans la poche. |
| Она положила письмо на стол. | Elle a posé la lettre sur la table. |

────────────────────────

Erreur fréquente

❌ **Положи вазу на стол** (debout)

✅ **Поставь вазу на стол.** — Mets le vase sur la table (debout).

❌ **Я ложу книгу**

✅ **Я кладу книгу.** — Je pose le livre.

────────────────────────

### Application réelle

**Bureau :** **Положите документы на стол, пожалуйста.** — Posez les documents sur la table, s'il vous plaît.

**Maison :** **Кладу ключи всегда на одно место.** — Je mets toujours les clés au même endroit.

**Magasin :** **Положите продукты в пакет.** — Mettez les produits dans le sac.

────────────────────────

🧠 À retenir

- **Класть / положить** = poser à l'horizontale.
- **Ставить / поставить** = mettre debout.
- **Accusatif** : **положить книгу**, **класть на стол**.
- Voir [Prendre : брать / взять](/manual/lecons/verbe-brat-vzyat).
