## Comment dire « nager », « naviguer » (direction) en russe ?

**Плыть / поплыть** = nager / naviguer dans une direction · **Плавать** = nager (multidirectionnel, loisir).

### Le mécanisme en une phrase

**Плыть** = nager / flotter vers un but (un trajet) · **Плавать** = nager (habitude, sans direction précise).

**Exemple :** **Лодка плывёт по реке.** — Le bateau navigue sur la rivière.

**Exemple :** **Поплыли!** — On s'est mis à nager ! / En route ! (bateau)

**Exemple :** **Я плаваю каждый день.** — Je nage tous les jours.

**Contre-exemple :** **Сейчас я плаваю через реку** — trajet : **плыву через реку**.

────────────────────────

Situation :

« On prend un bateau pour traverser la rivière à la datcha. »

↓

**Лодка уже плывёт — давай быстрее!** — Le bateau est déjà en route — dépêchons-nous !

────────────────────────

Pourquoi ?

**Плыть / плавать** complète **идти / ходить** pour l'eau.

```
  unidirectionnel              multidirectionnel
  ───────────────              ─────────────────
  плыть / поплыть              плавать
  плыву через реку             плаваю в бассейне
  корабль плывёт              люблю плавать
```

────────────────────────

💡 Astuce

**Плыть на + prépositionnel** (moyen) : **плыть на лодке** · **Плыть по + datif** : **плыть по реке**.

────────────────────────

⭐ Très fréquent

**Лодка плывёт.** · **Поплыли!** · **Умеешь плавать?**

────────────────────────

⚠ Attention

**Плыть** = eau · **Ехать** = véhicule terrestre · **Лететь** = voler.

────────────────────────

### Conjugaison

| | Плыть | Плавать |
|---|-------|---------|
| я | плыву | плаваю |
| ты | плывёшь | плаваешь |
| он/она | плывёт | плавает |

**Passé :** **плыл / плыла** · **плавал**

### Phrases utiles

| Russe | Français |
|-------|----------|
| Корабль плывёт в порт. | Le navire navigue vers le port. |
| Мы поплыли на лодке. | Nous sommes partis en bateau. |
| Он хорошо плавает. | Il nage bien. |
| Река быстро течёт — лодка плывёт. | La rivière coule vite — le bateau avance. |

────────────────────────

Erreur fréquente

❌ **Лодка плавает в порт** (direction)

✅ **Лодка плывёт в порт.**

❌ **Плыть в лодке**

✅ **Плыть на лодке.**

────────────────────────

### Application réelle

**Mer :** **Мы поплыли к острову.** — Nous avons nagé / navigué vers l'île.

**Piscine :** **Дети плавают в бассейне.** — Les enfants nagent à la piscine.

**Fleuve :** **По Волге плывут большие корабли.** — Sur la Volga naviguent de grands bateaux.

────────────────────────

🧠 À retenir

- **Плыть / поплыть** = nager / naviguer (direction).
- **Плавать** = nager (loisir, habitude).
- **На лодке**, **по реке** — prépositions clés.
