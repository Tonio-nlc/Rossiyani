## Que signifie **по** : « le long de », « par », « sur » ?

**По** est une préposition **polyvalente** : **по улице** (dans la rue / le long de la rue), **по русскому языку** (de russe / sur le russe), **по одному** (un par un). Selon le sens, elle prend le **datif** ou le **prépositionnel**.

### Le mécanisme en une phrase

**Mouvement le long de** : **по + datif** ( **по улице, по дороге** ).

**Thème / domaine** : **по + datif** ( **по математике, по телевизору** ).

**Distribution** : **по + datif** ( **по два, по одному** ).

**Exemple :** **Мы идём по улице.** — Nous marchons dans la rue.

**Contre-exemple :** **По улица** — **улица** au datif : **по улице**.

────────────────────────

Situation :

« Tu expliques comment rentrer chez toi à pied. »

↓

**ИДИТЕ ПРЯМО, ПОТОМ НАЛЕВО, ПО ЭТОЙ УЛИЦЕ ДО МЕТРО.** — Allez tout droit, puis à gauche, le long de cette rue jusqu'au métro.

────────────────────────

Pourquoi ?

**По** indique souvent un **parcours** ou une **direction étendue** (le long de…) — le datif marque la « surface » ou la « ligne » suivie. Pour les matières et sujets, **по** répond à « sur quel domaine ? » — comme « un cours **de** maths » en français.

```
По + datif

mouvement le long de → по улице
domaine / sujet → по математике
distribution → по одному
```

────────────────────────

💡 Astuce

**По телефону** (au téléphone) · **По почте** (par courrier) — expressions figées à mémoriser.

────────────────────────

⭐ Très fréquent

**По улице** · **По русски** (adverbe, sans nom) · **По одному.** — oral quotidien.

────────────────────────

⚠ Attention

**По-русски** (adverbe) ≠ **по русскому языку** (préposition + nom) — sens proche, forme différente.

────────────────────────

### Usages A2 fréquents

| Structure | Exemple | Sens |
|-----------|---------|------|
| **по + datif** (lieu) | **по городу** | dans la ville / à travers la ville |
| **по + datif** (sujet) | **экзамен по истории** | examen d'histoire |
| **по + datif** (distribution) | **по два яблока** | deux pommes chacun |
| **по + datif** (moyen) | **по телефону** | au téléphone |

**Exemple :** **У нас урок по математике.** — Nous avons cours de maths.

────────────────────────

Erreur fréquente

❌ **По улица** (nominatif)

✅ **По улице.**

❌ **Экзамен по математика**

✅ **Экзамен по математике.**

❌ **Иду по парк** (mouvement dans le parc)

✅ **Иду по парку.** (datif) · **Гуляю в парке.** (prépositionnel — être dans)

────────────────────────

### Application réelle

**Direction :** **Едьте по этой дороге до светофора.** — Roulez le long de cette route jusqu'au feu.

**Études :** **Я сдаю экзамен по русскому языку.** — Je passe l'examen de russe.

**Distribution :** **Раздайте по листу каждому.** — Distribuez une feuille à chacun.

────────────────────────

🧠 À retenir

- **По + datif** — le long de, domaine, distribution.
- **По улице, по дороге, по математике**.
- **По телефону, по почте** — expressions figées.
- Ne pas confondre **по-русски** (adverbe) et **по языку**.
- Voir [Prépositions в et на](/manual/lecons/prepositions-v-na).
