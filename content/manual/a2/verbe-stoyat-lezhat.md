## Comment dire « être debout », « être couché » en russe ?

**Стоять** = être debout · **Лежать** = être couché, être posé (objet) · verbes d'état, pas de mouvement.

### Le mécanisme en une phrase

**Стоять** = se tenir debout · **Лежать** = être allongé ou posé quelque part.

**Exemple :** **Он стоит у окна.** — Il est debout près de la fenêtre.

**Exemple :** **Книга лежит на столе.** — Le livre est posé sur la table.

**Exemple :** **Мы стояли в очереди час.** — Nous avons attendu debout dans la file une heure.

**Contre-exemple :** **Я лежу у окна** pour « je suis debout » — **я стою у окна**.

────────────────────────

Situation :

« On te demande où est ton sac ; il est posé sur la chaise. »

↓

**Сумка лежит на стуле.** — Le sac est sur la chaise.

────────────────────────

Pourquoi ?

**Стоять / лежать** décrivent positions stables — très fréquent à la maison, en ville, au travail.

```
  verbe           position          exemple
  ─────           ────────          ───────
  стоять          debout            стою у двери
  лежать          couché / posé     лежит на полу
  сидеть          assis             сижу на диване
```

────────────────────────

💡 Astuce

**Сидеть** = être assis — troisième verbe de position courant au A2.

────────────────────────

⭐ Très fréquent

**Стоять в очереди** · **Лежит на столе** · **Где лежит…?**

────────────────────────

⚠ Attention

**Лежать** pour les objets : **ключи лежат** (les clés sont posées) · **Стоять** pour les personnes debout.

────────────────────────

### Conjugaison

| | Стоять | Лежать |
|---|--------|--------|
| я | стою | лежу |
| ты | стоишь | лежишь |
| он/она | стоит | лежит |
| мы | стоим | лежим |
| вы | стоите | лежите |
| они | стоят | лежат |

**Passé :** **стоял / стояла** · **лежал / лежала**

### Phrases utiles

| Russe | Français |
|-------|----------|
| Он стоит на остановке. | Il est à l'arrêt (debout). |
| Где лежит мой телефон? | Où est mon téléphone ? |
| Машина стоит во дворе. | La voiture est garée dans la cour. |
| Документы лежат в папке. | Les documents sont dans le dossier. |

────────────────────────

Erreur fréquente

❌ **Книга стоит на столе** (livre posé à plat)

✅ **Книга лежит на столе.** — Le livre est sur la table.

❌ **Я лежу в очереди** (personne debout)

✅ **Я стою в очереди.** — Je suis dans la file (debout).

────────────────────────

### Application réelle

**Maison :** **Пасport лежит в ящике стола.** — Le passeport est dans le tiroir du bureau.

**Ville :** **Мы долго стояли в пробке.** — Nous avons longtemps été bloqués dans les embouteillages.

**Travail :** **Принтер стоит рядом с компьютером.** — L'imprimante est à côté de l'ordinateur.

────────────────────────

🧠 À retenir

- **Стоять** = être debout (personnes, véhicules arrêtés).
- **Лежать** = être couché ou posé (personnes, objets).
- **Сидеть** = être assis — complète le trio.
