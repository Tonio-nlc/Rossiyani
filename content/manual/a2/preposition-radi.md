## Comment dire « pour », « au nom de » (sacrifice, effort) en russe ?

**Ради + génitif** = pour, au nom de — souvent avec une nuance d'effort ou de dévouement.

### Le mécanisme en une phrase

**Ради + génitif** = pour (le bien de, en faveur de).

**Exemple :** **Ради детей.** — Pour les enfants.

**Exemple :** **Я сделал это ради тебя.** — Je l'ai fait pour toi.

**Exemple :** **Ради Бога, помогите!** — Pour l'amour du ciel, aidez !

**Contre-exemple :** **Ради подарок** — **ради подарка** (génitif).

────────────────────────

Situation :

« On explique qu'on accepte un travail difficile pour sa famille. »

↓

**Я работаю много ради семьи.** — Je travaille beaucoup pour ma famille.

────────────────────────

Pourquoi ?

**Ради** exprime motivation forte — plus émotionnel que **для** (but pratique).

```
  préposition    nuance              exemple
  ───────────    ──────              ───────
  для            but, usage          подарок для мамы
  ради           effort, dévouement  ради мамы
  из-за          cause négative      из-за дождя
  благодаря      grâce à             благодаря тебе
```

────────────────────────

💡 Astuce

**Ради Бога!** = expression figée (supplication) · **Ради приличия** = pour la forme.

────────────────────────

⭐ Très fréquent

**Ради детей** · **Ради тебя** · **Ради чего?**

────────────────────────

⚠ Attention

**Ради** + génitif uniquement : **ради мира**, **ради шутки** (pour rire).

────────────────────────

### Comparaison **для** vs **ради**

| Russe | Français | Nuance |
|-------|----------|--------|
| подарок для мамы | cadeau pour maman | objet destiné à |
| ради мамы | pour maman | effort, sacrifice |
| для работы | pour le travail | usage |
| ради интереса | par intérêt | motivation |

### Phrases utiles

| Russe | Français |
|-------|----------|
| Ради чего ты это делаешь? | Pourquoi tu fais ça ? (pour quoi ?) |
| Ради мира во всём мире. | Pour la paix dans le monde. |
| Он рискнул ради друга. | Il a pris un risque pour un ami. |
| Ради шутки сказал. | Il l'a dit pour rire. |

────────────────────────

Erreur fréquente

❌ **Ради дети**

✅ **Ради детей.** — Pour les enfants.

❌ Confondre **ради** et **для** dans tous les contextes

✅ **Книга для ребёнка** (objet) · **Ради ребёнка** (effort).

────────────────────────

### Application réelle

**Famille :** **Всё ради детей.** — Tout est pour les enfants.

**Travail :** **Я устал, но продолжаю ради проекта.** — Je suis fatigué, mais je continue pour le projet.

**Expression :** **Ради Бога, не шуми!** — Pour l'amour du ciel, ne fais pas de bruit !

────────────────────────

🧠 À retenir

- **Ради + génitif** = pour (effort, dévouement).
- Distinction **для** (but pratique) / **ради** (motivation forte).
- **Ради Бога** — expression figée.
- Voir [Для](/manual/lecons/preposition-dlya).
