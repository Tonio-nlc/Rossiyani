## Comment les verbes de mouvement s'emploient-ils au sens figuré ?

**Идти / ехать** au sens propre (marcher, rouler) · **Au figuré** : le temps passe, les choses évoluent, les événements « avancent ».

### Le mécanisme en une phrase

**Время идёт** · **Дела идут хорошо** · **Проект пошёл** = le projet a démarré / marche.

**Exemple :** **Время идёт быстро.** — Le temps passe vite.

**Exemple :** **Как идут дела?** — Comment vont les choses ?

**Exemple :** **Переговоры идут успешно.** — Les négociations se déroulent bien.

**Contre-exemple :** **Идти к успеху** (figuré) vs **идти домой** (littéral) — même verbe, contexte différent.

────────────────────────

Situation :

« On demande des nouvelles d'un projet en cours. »

↓

**Как идёт проект? — Пока всё идёт по плану.** — Comment avance le projet ? — Pour l'instant tout se passe comme prévu.

────────────────────────

Pourquoi ?

Usage **très fréquent** à l'oral — invisible dans les manuels débutants.

```
  littéral              figuré
  ────────              ──────
  идти домой            идти ко сну (s'endormir)
  ехать в Москву        дела идут
  идти пешком           время идёт
  навстречу + dat.      идти навстречу мечте
```

────────────────────────

💡 Astuce

**Как идут дела?** = **Как дела?** (variante) · **Пошло!** = ça marche ! (familier)

────────────────────────

⭐ Très fréquent

**Время идёт** · **Дела идут** · **Как идёт…?**

────────────────────────

⚠ Attention

**Идти** (unidirectionnel) au figuré souvent = progression · **Ходить** = répétition : **дела ходят** rare.

────────────────────────

### Exemples figurés

| Russe | Français |
|-------|----------|
| время идёт | le temps passe |
| дела идут хорошо | les choses vont bien |
| переговоры идут | les négociations avancent |
| идти ко сну | s'endormir |
| пошло! | ça marche ! |

────────────────────────

Erreur fréquente

❌ Traduire **время идёт** par « le temps marche »

✅ **Le temps passe.**

❌ **Ехать** pour « les choses vont » (sauf **дела идут**)

✅ **Идти** pour processus abstrait.

────────────────────────

### Application réelle

**Bureau :** **Совещание идёт уже два часа.** — La réunion dure déjà deux heures / se poursuit.

**Projet :** **Работа идёт полным ходом.** — Le travail avance à plein régime.

**Vie :** **Жизнь идёт своим чередом.** — La vie suit son cours.

────────────────────────

🧠 À retenir

- **Идти / ехать** = mouvement figuré courant.
- **Время идёт**, **дела идут**.
- **Как идёт…?** = comment avance… ?
