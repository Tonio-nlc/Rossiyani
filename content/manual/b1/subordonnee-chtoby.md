## Comment exprimer le but : **чтобы** ?

**Je t'appelle pour que tu saches**, **dis-le plus fort pour qu'on t'entende** — le russe utilise **чтобы + verbe au passé** pour le **but** ou la **volonté** dirigée vers autrui. Pas de subjonctif français : le **passé** remplit ce rôle.

### Le mécanisme en une phrase

**Action principale + , + чтобы + sujet + verbe au passé**.

**Exemple :** **Я позвонил, чтобы ты знал.** — J'ai appelé pour que tu saches.

**Exemple :** **Говори громче, чтобы все слышали.** — Parle plus fort pour que tout le monde entende.

**Contre-exemple :** **Чтобы ты знаешь** — après **чтобы**, le verbe va au **passé** : **знал / знала**.

────────────────────────

Situation :

« Tu expliques pourquoi tu envoies un message écrit. »

↓

**Пишу, чтобы не забыть.** — J'écris pour ne pas oublier.

────────────────────────

Pourquoi ?

**Чтобы** oriente l'action vers un **résultat voulu** — souvent pour **une autre personne**. Le passé (**знал, понял, услышал**) fonctionne comme le subjonctif français « pour qu'il sache » — c'est un **mode**, pas toujours un vrai passé.

```
But / intention

action
↓
чтобы
↓
passé (mode)
↓
Я позвонил, чтобы ты знал
```

────────────────────────

💡 Astuce

**Чтобы + infinitif** possible quand le sujet est **le même** : **Я пришёл, чтобы поговорить.** — Je suis venu pour parler. (même personne)

**Чтобы + passé** quand le sujet **change** : **Я пришёл, чтобы ты знал.**

────────────────────────

⭐ Très fréquent

**Скажи ему, чтобы он позвонил.** — Dis-lui de telephoner. (impératif indirect)

**Всё для того, чтобы…** — Tout cela pour que…

────────────────────────

Erreur fréquente

❌ **Чтобы он знает**

✅ **Чтобы он знал.**

❌ **Я учусь, чтобы сдать экзамен** — infinitif OK (même sujet)

✅ **Я учусь, чтобы сдать экзамен.** · **Я объяснил, чтобы он понял.**

❌ Oublier **что** dans **чтоб** — **чтоб** = forme courte orale de **чтобы**

✅ **Чтобы / чтоб** — oral accepte **чтоб**.

────────────────────────

⚠ Attention

**Для того, чтобы** = renforcement formel (« dans le but de »). **Чтобы** seul suffit à l'oral.

────────────────────────

### Application réelle

**Travail :** **Оставил заметку, чтобы коллега увидел.** — J'ai laissé un mot pour que le collègue le voie.

**Famille :** **Мы тихо говорим, чтобы ребёнок спал.** — Nous parlons bas pour que l'enfant dorme.

**Conseil :** **Возьми зонт, чтобы не промокнуть.** — Prends un parapluie pour ne pas te faire tremper.

────────────────────────

🧠 À retenir

- **Чтобы + passé** = pour que (+ subjonctif français).
- **Même sujet** : **чтобы + infinitif** possible.
- **Скажи ему, чтобы он…** — ordre indirect.
- **Для того, чтобы** — forme renforcée.
- Voir [Subordonnée потому что](/manual/lecons/subordonnee-potomu).
