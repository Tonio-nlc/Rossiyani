## Comment parler des animaux ?

**Un chien**, **deux chats**, **j'ai peur des chiens** — le vocabulaire animalier suit les règles déjà apprises : **genre**, **pluriel**, **есть/нет**, **бояться + génitif**.

### Le mécanisme en une phrase

Les noms d'animaux ont un **genre fixe** à mémoriser.

**Exemple :** **У меня есть собака.** — J'ai un chien. **собака** = féminin ( même si mâle ).

**Exemple :** **У соседа два кота.** — Le voisin a deux chats. **два кота** (génitif singulier après 2).

**Contre-exemple :** **У меня есть собак** — après **есть**, le nom reste au **nominatif** singulier : **есть собака**.

────────────────────────

Situation :

« On te demande si tu as un animal à la maison. »

↓

**У МЕНЯ ЕСТЬ СОБАКА.** — **есть + nominatif** · **нет + génitif**.

────────────────────────

Pourquoi ?

En russe, le **nom d'animal** ne change pas selon le sexe biologique dans la conversation courante : **собака** peut désigner un chien mâle ou femelle. Pour préciser, on ajoute **мальчик / девочка**. **Бояться + génitif** revient souvent : **боюсь собак**, **боюсь этой собаки**.

```
Animaux

есть + nominatif (avoir)
нет + génitif (ne pas avoir)
бояться + génitif (avoir peur)
↓
есть собака / нет кошки / боюсь собак
```

────────────────────────

💡 Astuce

**Собака** = chien (féminin fixe) · **кот** = chat mâle · **кошка** = chat (féminin).

────────────────────────

⭐ Très fréquent

**У меня есть…** · **У нас дома кошка.** · **Боюсь собак.** — possession et peur.

────────────────────────

⚠ Attention

**Боюсь собака** = incorrect. **Боюсь собак** — génitif pluriel après **бояться**.

────────────────────────

**Exemple :** **Моя кошка чёрная.** — Mon chat est noir.

**Exemple :** **На улице много собак.** — Il y a beaucoup de chiens dans la rue.

────────────────────────

Erreur fréquente

❌ **Два кошки** (pluriel nominatif)

✅ **Две кошки.** (**две** + gén. sing. ou **две кошки** — **две** féminin + **кошки** gen. sing.)

Actually: два кота, две кошки - correct

❌ **У меня нет собаки** vs **нет собаки** — les deux existent ; **нет + génitif** : **нет собаки** (je n'ai pas de chien)

✅ **У меня нет собаки.**

❌ **Боюсь собака** (nominatif)

✅ **Боюсь собак.** / **Боюсь этой собаки.**

────────────────────────

### Application réelle

**Maison :** **У нас дома кошка, её зовут Муся.** — Chez nous il y a un chat, elle s'appelle Mousya.

**Allergie :** **Извините, я боюсь собак.** — Désolé, j'ai peur des chiens.

**Promenade :** **Можно с собакой?** — Les chiens sont-ils admis ?

────────────────────────

🧠 À retenir

- **Собака, кошка** — animaux domestiques fréquents.
- **Есть / нет + nom** ( **есть собака**, **нет кошки** ).
- **Бояться + génitif**.
- Comptage : **две кошки, три собаки, пять собак**.
- Voir [Sentiments](/manual/lecons/sentiments-chuvstva) (**боюсь**).
