## Comment exprimer « pour » (à l'intention de) ?

« Un cadeau **pour** Anna », « c'est **pour** toi », « une table **pour** deux » — le sens « pour / à destination de » en russe passe par **для + génitif**. Le nom ou pronom indique **pour qui** ou **pour quoi** quelque chose est destiné.

Le **génitif** est la forme déjà utilisée avec **нет**, **у**, **из**, **без** — ici il marque le **bénéficiaire** ou le **destinataire**.

### Le mécanisme en une phrase

**Для + génitif** = à l'intention de / pour.

**Exemple :** **Это для тебя.** — C'est pour toi. **Ты** ne reste pas au nominatif : on dit **для тебя** (génitif du pronom).

**Exemple :** **Подарок для мамы.** — Un cadeau pour maman. **Мама** devient **мамы**.

**Contre-exemple :** **для мама** — nominatif interdit après **для**.

────────────────────────

Situation :

« Tu offres un cadeau et tu précises pour qui c'est. »

↓

**ЭТО ДЛЯ ТЕБЯ. — НЕТ, ДЛЯ НЕЁ.** — **для тебя / для неё** (génitif des pronoms).

────────────────────────

Pourquoi ?

**Для** oriente l'objet ou l'action **vers le bénéficiaire** sans mouvement physique obligatoire. On ne « va pas vers » la personne (**к**), on désigne **pour qui** l'objet est pensé. Le génitif encode cette relation « de / pour le compte de ».

**Для детей** — pour les enfants.

**Для работы** — pour le travail.

```
Для + génitif

bénéficiaire / objectif
↓
для тебя / для мамы / для работы
```

⭐ Très fréquent

**Это для тебя.** · **Подарок для мамы.** · **Для работы.** — cadeaux et usage.

⚠ Attention

**Для** (pour) ≠ **к** (vers) : **подарок для друга** vs **иду к другу**.

────────────────────────

### Formes fréquentes

| Contexte | Russe | Traduction |
|----------|-------|------------|
| Pronoms | **для меня, для тебя, для него, для неё, для нас, для вас, для них** | pour moi, toi… |
| Personnes | **для друга, для Анны, для детей** | pour un ami, Anna, les enfants |
| Objectif | **для учёбы, для отдыха** | pour les études, le repos |

**Для + numéral :** **стол для двоих** — table pour deux (forme spéciale **двоих**).

────────────────────────

Erreur fréquente

❌ **для ты**

✅ **для тебя**

❌ **для друг** (nominatif)

✅ **для друга**

❌ Confondre **для** (pour) et **к** (vers) : **иду к другу** (je vais chez lui) vs **подарок для друга** (cadeau pour lui).

✅ **к другу** / **для друга**

────────────────────────

### Application

**Cadeau :** **Я купил цветы для мамы.** — J'ai acheté des fleurs pour maman.

**Usage :** **Этот телефон для работы, не для игр.** — Ce téléphone est pour le travail, pas pour jouer.

**Service :** **Меню на русском для иностранцев.** — Menu en russe pour les étrangers.

**Question :** **Это для вас? — Нет, для неё.** — C'est pour vous ? — Non, pour elle.

────────────────────────

🧠 À retenir

- **Для + génitif** = pour (destinataire, objectif).
- Pronoms : **для меня, для тебя, для него, для неё…**
- Ne pas confondre **для** (pour) et **к** (vers une personne).
- Mêmes terminaisons génitif que **нет**, **без**, **из**.
- Très fréquent : **для детей, для работы, для тебя**.
