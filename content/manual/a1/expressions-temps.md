## Comment dire « bientôt », « déjà », « encore » ?

**Сейчас** (maintenant), **скоро** (bientôt), **уже** (déjà), **ещё** (encore / pas encore) — quatre adverbes de temps **invariables** qui cadrent presque toute phrase au quotidien.

### Le mécanisme en une phrase

**Сейчас** = maintenant · **Скоро** = bientôt · **Позже** = plus tard.

**Уже** = déjà (action accomplie ou plus tôt que prévu).

**Ещё** = encore / toujours · **Ещё не** = pas encore.

**Exemple :** **Я уже дома.** — Je suis déjà à la maison.

**Contre-exemple :** **Я ещё дома** pour « je suis déjà à la maison » — **ещё** = encore : **Я ещё не дома.** (pas encore) vs **Я уже дома.** (déjà).

────────────────────────

Situation :

« On te demande si tu as fini ton travail. »

↓

**УЖЕ ЗАКОНЧИЛ. СКОРО БУДУ.** — J'ai déjà fini. J'arrive bientôt.

────────────────────────

Pourquoi ?

Ces adverbes ne se conjuguent pas — ils se placent souvent **avant le verbe** ou en **début de phrase** pour marquer le moment. **Уже** et **ещё** forment un couple opposé (comme « déjà » vs « encore / pas encore » en français).

```
Temps oral

сейчас / скоро / позже
уже ↔ ещё (не)
↓
Я уже дома. / Я ещё не готов.
```

────────────────────────

💡 Astuce

**Ещё раз** = encore une fois · **Уже скоро** = bientôt maintenant — renforcement fréquent.

────────────────────────

⭐ Très fréquent

**Скоро увидимся!** · **Я уже ем.** · **Ещё рано.** — rendez-vous et quotidien.

────────────────────────

⚠ Attention

**Ещё** + négation : **ещё не** — pas encore. **Не ещё** = incorrect.

────────────────────────

### Tableau A1

| Russe | Français |
|-------|----------|
| **сейчас** | maintenant |
| **скоро** | bientôt |
| **позже** | plus tard |
| **потом** | ensuite / après |
| **уже** | déjà |
| **ещё** | encore / toujours |
| **ещё не** | pas encore |
| **рано** | tôt |
| **поздно** | tard |

**Exemple :** **Сейчас поздно, давай завтра.** — Il est tard maintenant, faisons ça demain.

────────────────────────

Erreur fréquente

❌ **Я ещё готов** pour « je suis déjà prêt »

✅ **Я уже готов.**

❌ **Скоро я буду скоро** (redondance)

✅ **Я скоро буду.** / **Скоро приду.**

❌ **Не уже** pour « pas encore »

✅ **Ещё не.**

────────────────────────

### Application réelle

**Retard :** **Извините, я опоздаю. Буду через десять минут.** — Désolé, je suis en retard. J'arrive dans dix minutes.

**Attente :** **Ты уже готов? — Ещё нет, подожди.** — Tu es prêt ? — Pas encore, attends.

**Plan :** **Сначала работа, потом отдых.** — D'abord le travail, ensuite le repos.

────────────────────────

🧠 À retenir

- **Сейчас, скоро, позже, потом** — cadrage temporel.
- **Уже** = déjà · **Ещё** = encore · **Ещё не** = pas encore.
- Adverbes invariables, souvent avant le verbe.
- **Рано / поздно** — trop tôt / trop tard.
- Voir [Heures et temps](/manual/lecons/heures-temps).
